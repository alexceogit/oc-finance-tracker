## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## PWA Features

This app is a Progressive Web App that can:
- Be installed on mobile and desktop
- Work offline
- Send to home screen on mobile devices
